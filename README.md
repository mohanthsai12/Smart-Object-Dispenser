# Smart-Object-Dispenser
An IoT-based Smart Object Dispenser that automatically dispenses products after successful UPI payment verification. The system integrates a web application, Firebase Realtime Database, and an ESP32 microcontroller to provide a real-time, cashless vending solution.

Project Overview

The Smart Object Dispenser is an IoT-enabled automated vending solution that combines embedded systems, cloud computing, and digital payment technologies to provide a secure, cashless, and real-time dispensing experience. The project addresses the limitations of traditional vending machines by offering a compact, affordable, and scalable system suitable for educational institutions, offices, and small commercial environments. The solution integrates a web-based user interface, Firebase Realtime Database, and an ESP32 microcontroller to automate the complete dispensing workflow.

Users can browse available products through a web application, add items to a shopping cart, and complete transactions using UPI payment methods such as Google Pay, PhonePe, or Paytm. Once the payment is confirmed, transaction details are synchronized with Firebase Realtime Database, enabling seamless communication between the cloud and the embedded hardware. The ESP32 continuously monitors the database and automatically activates the corresponding servo motor and DC motor to dispense the selected product. After successful dispensing, the system updates the transaction status, ensuring reliable and transparent operation.

The project demonstrates practical implementation of IoT architecture, firmware development, cloud integration, embedded automation, and real-time hardware-software communication. It incorporates technologies including ESP32, Firebase Realtime Database, HTML, JavaScript, Arduino IDE, Wi-Fi communication, REST APIs, SG90 servo motors, L298N motor driver, relay modules, and JSON-based data exchange. These technologies collectively enable low-latency communication and dependable system performance.

This project significantly enhanced my expertise in embedded systems, IoT application development, cloud-connected automation, firmware programming, API integration, and system debugging. It also strengthened my understanding of designing scalable smart devices capable of integrating modern digital payment systems with real-time hardware control. The modular architecture allows future enhancements such as inventory management, automated payment verification, mobile application support, analytics dashboards, and remote monitoring, making it a practical foundation for next-generation smart vending and automation systems.
